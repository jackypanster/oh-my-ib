# Card 01 — omi pnl (account-level PnL)

Make `tests/pnl_command.rs` green: add the read-only `omi pnl` subcommand reporting account-level
Daily/Unrealized/Realized PnL, plus the pure sentinel filter `pnl_number`. Read `arch.md` + ADR 0007 first.

## Scope (impl-paths only — do NOT touch `tests/pnl_command.rs`)

- `src/ib/pnl.rs` (new):
  - `pub fn pnl_number(raw: Option<f64>) -> serde_json::Value` — the pure, frozen seam:
    `match raw { Some(x) if x.is_finite() && x.abs() != f64::MAX => Value::from(x), _ => Value::Null }`.
    (IB encodes "no value" as `Double.MAX_VALUE == f64::MAX`; that + any non-finite ⇒ `null`. Decision A.)
  - `pub fn pnl(cfg: &Config) -> Result<serde_json::Value, AppError>` — mirror `account.rs`:
    `let client = super::connect(cfg)?; let account = super::resolve_account(&client, cfg)?;`
    `let sub = client.pnl(&account, None).map_err(|e| AppError::data(format!("pnl request failed: {e}"), "pnl"))?;`
    Take EXACTLY ONE reading via `sub.next_data()` (ADR 0007 — reqPnL is unbounded, NO End marker; a
    `for … in sub.iter_data()` loop HANGS):
      - `Some(Ok(p))` → build the object below.
      - `Some(Err(e))` → `AppError::data(format!("pnl stream: {e}"), "pnl")`.
      - `None` → `AppError::data("no PnL reading", "pnl")`.
    Object: `json!({ "account": account.0, "daily_pnl": pnl_number(Some(p.daily_pnl)),
      "unrealized_pnl": pnl_number(p.unrealized_pnl), "realized_pnl": pnl_number(p.realized_pnl) })`.
    (`ibapi::accounts::PnL { daily_pnl: f64, unrealized_pnl: Option<f64>, realized_pnl: Option<f64> }`.)
- `src/ib/mod.rs`: `mod pnl;` + `pub use pnl::{pnl, pnl_number};`.
- `src/cli.rs`: add `Pnl` to `enum Command` (no args; doc-comment like the other read commands).
- `src/main.rs`: dispatch `Command::Pnl => ib::pnl(&config),`.

Do NOT touch `src/output.rs` — `--format table` is free (generic `render_table`). No new dependency
(`ibapi` already a dep; the seam needs none). `spec-paths ∩ impl-paths = ∅`.
