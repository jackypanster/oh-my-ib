# Card 01 — live-write-guardrail (live write posture guardrail)

`CONTEXT.md` first — `arch.md §Write-set` is the verbatim implementation. WRITE-path card in
`src/ib/trade.rs`; the port gate `require_live_write_gate` is EXTENDED alongside, never weakened.

## Scope (impl-paths only — do NOT touch `tests/`)

- `src/ib/trade.rs` — add four NEW pure `pub fn` seams + one const (exact signatures in `arch.md`):
  - `compute_notional(quantity: f64, limit: Option<f64>, multiplier: f64) -> Option<f64>`
    = `limit.map(|l| quantity * l.abs() * multiplier)` (mirror `shape_preview`, line 85).
  - `resolve_max_notional(raw: Option<&str>) -> Result<f64, String>`
    = `None ⇒ Ok(DEFAULT_MAX_NOTIONAL)`; `Some(s) ⇒` `s.trim().parse::<f64>()`, require `is_finite()` ∧
    `> 0.0`, else `Err("invalid OMI_MAX_NOTIONAL '{s}': expected a positive number")`.
  - `check_live_write_posture(is_live: bool, is_mkt: bool, notional: Option<f64>, cap: f64) -> Result<(), String>`
    = `!is_live ⇒ Ok(())`; `is_mkt ⇒ Err("live orders must be LMT — pass --limit (MKT is paper-only)")`;
    `match notional { Some(n) if n > cap ⇒ Err("live notional {n} exceeds cap {cap} — raise OMI_MAX_NOTIONAL to override"), _ ⇒ Ok(()) }`.
  - `refuse_live_combo_on_live(is_live: bool) -> Result<(), String>`
    = `if is_live { Err("option-combo is paper-only during the trial (interlock posture) — use paper :4002".into()) } else { Ok(()) }`.
  - `pub const DEFAULT_MAX_NOTIONAL: f64 = 500.0;`
- `src/ib/trade.rs` — WIRE (review-by-reading; ADR 0030 §Seams):
  - `place_core` (line 468): after the `if cfg.preview { … }` branch and the existing
    `require_live_write_gate(cfg)?`, BEFORE `connect`, derive `is_live = cfg.port == LIVE_PORT`,
    `cap = resolve_max_notional(std::env::var("OMI_MAX_NOTIONAL").ok().as_deref()).map_err(|m| AppError::config(m, ctx))?`,
    `multiplier = if matches!(contract.security_type, SecurityType::Option) { 100.0 } else { 1.0 }`,
    `is_mkt = order.order_type == "MKT"`, `notional = compute_notional(order.total_quantity, order.limit_price, multiplier)`;
    then `check_live_write_posture(is_live, is_mkt, notional, cap).map_err(|m| AppError::config(m, ctx))?;`.
  - `option_combo` (site line 766): inside the `if !cfg.preview { … }`, BEFORE `require_live_write_gate`,
    add `refuse_live_combo_on_live(cfg.port == LIVE_PORT).map_err(|m| AppError::config(m, ctx))?;`.
- `src/ib/mod.rs` (line 45): add the four fns to the existing `pub use trade::{…}` re-export.
