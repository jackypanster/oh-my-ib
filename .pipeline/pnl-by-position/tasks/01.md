# Card 01 — omi pnl-by-position (per-position PnL sweep)

Make `tests/pnl_by_position_command.rs` green: add the read-only `omi pnl-by-position` subcommand —
per-position Daily/Unrealized/Realized PnL via a discovery pass + N sequential `pnl_single` take-first
reads — plus the pure shaping seam `shape_pnl_by_position`. Read `arch.md` + `CONTEXT.md` +
**ADR 0009** (and ADR 0007 it builds on) first.

## Scope (impl-paths only — do NOT touch `tests/pnl_by_position_command.rs`)

- `src/ib/pnl_by_position.rs` (new):
  - **Plain, ibapi-free row** (the frozen test constructs these — fields must match exactly):
    ```
    pub struct PnlSingleRow { pub conid: i32, pub symbol: String, pub position: f64,
      pub daily_pnl: f64, pub unrealized_pnl: f64, pub realized_pnl: f64, pub value: f64 }
    ```
  - `pub fn shape_pnl_by_position(rows: Vec<PnlSingleRow>) -> serde_json::Value` — the pure, frozen
    shaping seam. For each row **in order**, emit an object with exactly 7 keys:
    `conid`, `symbol`, `position` raw (`Value::from`); `daily_pnl`, `unrealized_pnl`, `realized_pnl`,
    `value` each through `super::pnl_number(Some(x))` (sentinel `f64::MAX` / non-finite → `null`).
    Return `Value::Array(rows)`; empty input → `json!([])`. Never filter rows (qty==0 is data — PRD D6).
  - `pub fn pnl_by_position(cfg: &Config) -> Result<serde_json::Value, AppError>` — the gateway fn
    (NOT frozen; arch.md §Components step-by-step):
    1. `let client = super::connect(cfg)?; let account = super::resolve_account(&client, cfg)?;`
    2. **Discovery** (mirror `positions.rs`): `client.account_updates(&account)`, iterate
       `iter_data()`, collect `(p.contract.contract_id, p.contract.symbol)` from every
       `AccountUpdate::PortfolioValue(p)`, `break` on `AccountUpdate::End`; map stream errors to
       `AppError::data(..., "pnl-by-position")`. **Drop the subscription before the sweep**
       (explicit `drop(subscription)` — Drop sends the cancel).
    3. **Sweep** (ADR 0009): per `(conid, symbol)` in discovery order:
       `client.pnl_single(&account, ContractId::from(conid), None)` (import
       `ibapi::accounts::types::ContractId`) → **take-first** via `next_data()` (NEVER `iter_data()`
       — markerless stream, a drain loop hangs forever): `Some(Ok(p))` → build `PnlSingleRow` with
       `symbol` from discovery and `position`/`daily_pnl`/`unrealized_pnl`/`realized_pnl`/`value`
       from the reading; `Some(Err(e))`/`None` → **fail-fast**
       `AppError::data(format!("pnl_single conid {conid}: ..."), "pnl-by-position")` — no partial
       output. Drop each subscription after its reading (scope exit suffices).
    4. `Ok(json!({ "account": account.0, "by_position": shape_pnl_by_position(rows) }))`.
- `src/ib/mod.rs`: `mod pnl_by_position;` +
  `pub use pnl_by_position::{pnl_by_position, shape_pnl_by_position, PnlSingleRow};`.

- `src/main.rs`: dispatch `Command::PnlByPosition => ib::pnl_by_position(&config),`.
