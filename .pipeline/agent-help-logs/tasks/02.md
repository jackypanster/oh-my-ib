## Steps

     via `std::env::var_os("HOME")` (mirror `Config::config_path()`, src/config.rs:73).
   - `AuditEntry` (serde Serialize): `ts` (RFC3339 UTC, `time` crate — already a dep),
     `cmd` (`surface::command_name()`), `argv` (`std::env::args().skip(1)`, the value

     `exit` (i32), `error` (Option<&'static str> = `AppError::code()`), `duration_ms`.
   - `append(&AuditEntry) -> Result<..>`: `create_dir_all` parent + append one JSON line.
   - `read_tail(n) -> (Vec<serde_json::Value>, usize)`: last n parsed entries (newest
     last) + skipped-malformed count; missing file ⇒ `(vec![], 0)`.
2. `src/lib.rs`: `pub mod audit;`.
3. `src/main.rs` — the seam (arch §Data flow): after parse succeeds, `Instant::now()`;
   run; derive exit/error; build entry; `audit::append` FAIL-OPEN — on write error print
   ONE plain line to stderr starting `warn:` (e.g. `warn: audit log write failed: ...`),
   NEVER the JSON error envelope, NEVER a changed exit code. Then emit + exit as today.
   Help/Logs are audited too (uniform seam; no special cases).
4. Replace card 01's placeholder `Command::Logs(args)` arm: BEFORE `Config::load()`,
   return `{"path": <log_path>, "entries": [...read_tail(args.tail)...],
   "skipped_malformed": n}`.
5. `cargo build && cargo test --test logs_command` green;
   `cargo clippy --all-targets -- -D warnings` clean; commit(s) on `feat/agent-help-logs`.
