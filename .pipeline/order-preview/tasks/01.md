# Card 01 — order-preview (whatIf preview for the write path)

Make `tests/order_preview_command.rs` GREEN. Read `PRD.md` + `arch.md` + `CONTEXT.md` + ADR 0026
FIRST — arch.md §Data flow + §The two new seams is the verbatim implementation. Money-adjacent:
the preview path shares the write gate; do NOT weaken it.

## Scope (impl-paths ONLY — do NOT touch `tests/`)

- **`src/cli.rs`** — add to `GlobalOpts` (mirror `pub live: bool`, cli.rs:33):
  `#[arg(long, global = true)] pub preview: bool`. Doc: "Preview the order via IB whatIf (no
  transmit): resolved contract + margin/commission, no order placed".
- **`src/config.rs`** — add `pub preview: bool` to `Config` (default `false` in `Config::default`);
  in `merge_flags` add `self.preview = g.preview;` (CLI-only, no config-file key).
- **`src/ib/trade.rs`** (the ONLY file with write/gateway calls):
  - `pub fn shape_preview(contract: &Contract, order: &Order, state: &OrderState) -> Value` — PURE
    FROZEN seam, mirrors `shape_order_ack` (trade.rs:54). Emit the exact uniform envelope:
    ```
    { "preview": true, "what_if": true, "action": <{:?} of order.action>,
      "contract": { "symbol": <contract.symbol.to_string()>, "sec_type": <STK|OPT|BAG…>, "conid": … },
      "order": { "type": order.order_type, "qty": order.total_quantity, "limit": order.limit_price },
      "margin": { "init_change": state.initial_margin_change, "maint_change": state.maintenance_margin_change,
                  "equity_with_loan_change": state.equity_with_loan_change },
      "commission": { "value": state.commission, "min": state.minimum_commission,
                      "max": state.maximum_commission, "currency": state.commission_currency },
      "warning": state.warning_text, "status": <{:?} of state.status> }
    ```
    `Option<f64>` fields serialize to JSON `null` when `None` (serde_json does this via `json!`) —
    the frozen test asserts key-present + null. Keys are EXACT (frozen `keys` array).
  - `fn preview_with_client(client, ctx, contract, order, account) -> Result<Value, AppError>` —
    gateway fn (review-by-reading; NOT frozen). = `place_with_client` (trade.rs:317) with two
    deltas: after the account stamp, set `order.what_if = true`; in the bounded first-ack loop
    return `shape_preview(contract, &order, &od.order_state)` on the first `PlaceOrder::OpenOrder(od)`
    (the whatIf payload rides `OpenOrder.order_state`; skip `OrderStatus`/other events). Keep the same
    `TAKE_FIRST_TIMEOUT` + UNKNOWN-envelope semantics; for preview UNKNOWN just means "no preview
    data" (nothing transmitted, assuming Tiger honors whatIf).
  - **Branch (the only new control flow), AFTER the gate:** in `place_core` (trade.rs:381), after
    `resolve_account`, add `if cfg.preview { return preview_with_client(&client, ctx, contract, order, &account); }`
    before the existing `place_with_client` call. In `option_combo` (trade.rs:~726) and
    `option_close` (trade.rs:~974) — which call `place_with_client` directly, each already AFTER
    their own `require_live_write_gate` — swap the call for
    `if cfg.preview { preview_with_client(...) } else { place_with_client(...) }`. The real
    (`what_if=false`) path MUST stay byte-identical — the frozen stk/option/combo/close suites assert it.
- **`src/ib/mod.rs`** — add `shape_preview` to the `pub use trade::{…}` re-export (mod.rs:45).
