## Scope (impl-paths only — do NOT touch `tests/`)

- `src/cli.rs`: `OptionQuote(OptionQuoteArgs)` variant; `OptionQuoteArgs` verbatim from
  arch.md §CLI (all flags; `--exchange` default `"SMART"`, `--currency` default `"USD"`,
  `--trading-class` Option). Doc-comment: "Snapshot quote + greeks for one option contract".
- `src/ib/option_quote.rs` (NEW):
  - `pub struct GreeksRow { ... 7 × pub Option<f64>: implied_volatility, delta, gamma, vega,
    theta, option_price, underlying_price }` + `#[derive(Default)]` (frozen tests use
    `..Default::default()`); ibapi-free construction in tests.
  - `pub fn option_quote_greeks(tick: &TickTypes) -> Option<GreeksRow>` — pure FROZEN seam:
    `Some` ONLY for `TickTypes::OptionComputation` with `field` ∈
    {`TickType::ModelOption`, `TickType::DelayedModelOption`}; map the 7 value fields;
    everything else (Bid/Ask/Last/CustOptionComputation rows, price/size ticks) ⇒ `None`.
  - `pub fn shape_option_quote(symbol: &str, expiry: &str, strike: f64, right: &str,
    exchange: &str, currency: &str, trading_class: Option<&str>, delayed: bool,
    ticks: serde_json::Map<String, Value>, greeks: Option<GreeksRow>) -> Value` — pure
    FROZEN seam: 8-key `contract` echo (`multiplier` always `"100"`; `trading_class` null
    when None); `right` normalized case-insensitively to `"C"`/`"P"` (accepts c/C/call/CALL/
    p/P/put/PUT — validation upstream guarantees membership); `greeks` key present IFF
    `Some`, inside it ONLY Some-valued fields; `ticks` pass-through; `delayed` echoed.
    (Signature refined from arch.md: seam decoupled from the clap struct — the test is the
    contract.)
  - `pub fn option_quote(cfg: &Config, args: &OptionQuoteArgs) -> Result<Value, AppError>` —
    gateway fn (NOT frozen): pre-connect validation (`right` parses to Call/Put else
    `AppError::usage`; `strike > 0.0` else usage; `expiry` = 8 ASCII digits, m∈1..=12,
    d∈1..=31 else usage) → connect → `switch_market_data_type` (quote.rs:32-39 verbatim,
    same `delayed` threading) → `Contract::call(..)`/`::put(..)` `.strike(..)`
    `.expires_on(y, m, d)` `.on_exchange(..)` `.in_currency(..)` [`.trading_class(..)` when
    Some] `.build()` → `market_data(&contract).snapshot().subscribe()` → bare `iter_data()`
    to `SnapshotEnd` (quote.rs class, deliberately NOT timeout-wrapped — ADR 0019 D2):
    `quote_price_tick` hits → `ticks`; `option_quote_greeks` hits → replace greeks
    (last-model-row-wins); error context `"option-quote"`.
- `src/ib/mod.rs`: `mod option_quote;` +
  `pub use option_quote::{option_quote, option_quote_greeks, shape_option_quote, GreeksRow};`
- `src/main.rs`: one dispatch arm.
