# Card 01 — omi executions (current-day fills)

Make `tests/executions_command.rs` green: add the read-only `omi executions` subcommand reporting the
account's **current-day** executions (fills), joined to their commission reports by `exec_id`, plus the
pure JOIN seam `merge_executions`. Read `arch.md` + `CONTEXT.md` + **ADR 0008** first.

## Scope (impl-paths only — do NOT touch `tests/executions_command.rs`)

- `src/ib/executions.rs` (new):
  - **Plain, ibapi-free rows** (the frozen test constructs these — fields must match exactly):
    ```
    pub struct ExecRow { pub exec_id: String, pub order_id: i32, pub perm_id: i64, pub time: String,
      pub symbol: String, pub conid: i32, pub side: String, pub shares: f64, pub price: f64,
      pub cumulative_qty: f64, pub avg_price: f64, pub exchange: String }
    pub struct CommissionRow { pub exec_id: String, pub commission: f64, pub currency: String,
      pub realized_pnl: Option<f64> }
    ```
  - `pub fn merge_executions(execs: Vec<ExecRow>, comms: Vec<CommissionRow>) -> serde_json::Value` — the
    pure, frozen JOIN seam. Index `comms` by `exec_id` (e.g. `HashMap<String,&CommissionRow>`, last write
    wins). For each `ExecRow` **in order**, emit an object with the passthrough fields
    (`exec_id, order_id, perm_id, time, symbol, conid, side, shares, price, cumulative_qty, avg_price,
    exchange`) plus, if a commission matches:
    `commission = Value::from(c.commission)`, `commission_currency = Value::from(c.currency)`,
    `realized_pnl = super::pnl_number(c.realized_pnl)`; else all three = `Value::Null`. Orphan commissions
    (exec_id matching no execution) are **not** emitted. Return `Value::Array(rows)`.
  - `pub fn executions(cfg: &Config) -> Result<serde_json::Value, AppError>` — the gateway fn (mirror
    `orders.rs`/`positions.rs` drain, NOT `pnl.rs` take-first):
    `let client = super::connect(cfg)?; let account = super::resolve_account(&client, cfg)?;`
    build `ExecutionFilter { account_code: account.0.clone(), ..Default::default() }`;
    `let sub = client.executions(filter).map_err(|e| AppError::data(format!("executions request failed: {e}"), "executions"))?;`
    **Drain to End** via `for item in sub.iter_data()` (ADR 0008 — `ExecutionDataEnd → EndOfStream`, so the
    loop ends naturally; a `for … in iter_data()` is correct here, unlike reqPnL). Map each item
    (`item.map_err(|e| AppError::data(format!("executions stream: {e}"), "executions"))?`):
      - `Executions::ExecutionData(d)` → push an `ExecRow` from `d.contract.symbol`, `d.contract.contract_id`
        (→ `conid`) and `d.execution` fields (`execution_id`→`exec_id`, `order_id`, `perm_id`, `time`,
        `exchange`, `side: d.execution.side.as_str().to_string()` → "BOT"/"SLD", `shares`, `price`,
        `cumulative_quantity`→`cumulative_qty`, `average_price`→`avg_price`).
      - `Executions::CommissionReport(c)` → push a `CommissionRow` (`execution_id`→`exec_id`, `commission`,
        `currency`, `realized_pnl`).
    Then `Ok(json!({ "account": account.0, "executions": merge_executions(execs, comms) }))`.
    (ibapi imports: `ibapi::orders::{Executions, ExecutionFilter}`; the item variants carry
    `ExecutionData`/`CommissionReport`.)
- `src/ib/mod.rs`: `mod executions;` + `pub use executions::{executions, merge_executions, ExecRow, CommissionRow};`.
- `src/cli.rs`: add `Executions` to `enum Command` (no args; doc-comment like the other read commands).
- `src/main.rs`: dispatch `Command::Executions => ib::executions(&config),`.

Do NOT touch `src/output.rs` — `--format table` is free (generic `render_table`; `null` renders as `null`).
No new dependency (`ibapi` already a dep; the seam needs none — `pnl_number` is already re-exported).
