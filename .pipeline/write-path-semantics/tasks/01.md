## Do (all in `docs/write-path-semantics.md`)

1. Write an agent-first (dense, machine-parseable; AGENTS.md §Authoring) doc with a **7-column table**:
   `field | our value | ibapi type/behavior | Tiger/IB reference semantics | why this value / deliberate
   divergence | boundary cases | verification tier`.
2. **Rows** (grouped; ~15–20 total):
   - *Explicitly-set Order fields*: `action`, `total_quantity`, `order_type` (MKT/LMT), `tif` (Day),
     `limit_price` (single-leg vs combo), `account` (choke-point stamp).
   - *Load-bearing ibapi Default fields* (silently sent via `..Default::default()`, source
     `orders/mod.rs:478`): `transmit`, `outside_rth`, `display_size`, `what_if`, `origin`, `exempt_code`.
   - *Contract surface* (`Contract::stock/call/put/spread`, `contracts/builders.rs`): `symbol`,
     `security_type` (STK/OPT/BAG), `exchange` (SMART), `currency` (USD), `multiplier` (100 for options),
     `strike`, `right`.

   - *Inert tail*: ONE catch-all row — "all remaining ~70 `Order` fields inherit ibapi Default = inert
     (empty/zero/`None`/`vec![]`); not send-meaningful."
3. **Verification tier** per row — exactly one of:
   - `✅ paper-probe` — pre-fill **`account` = ✅** (order-account-stamp probe, journal seq=4/6: Tiger
     accepts an explicit `Order.account`; PreSubmitted paper order corroborates `transmit` transmits).
   - `📖 doc-cite` — cite an ibapi source line and/or the IB TWS API Order/Contract field reference
     (e.g. `transmit` ← `mod.rs:494` + IB "transmit=false stages the order"; SMART/USD/mult-100 ←
     `builders.rs:23-24,93-95`).
   - `⚠️ UNVERIFIED` — at minimum the combo net-credit sign and `display_size: Some(0)` (the crate itself
     carries `// TODO - default to None?`).
4. **`## ⚠️ Risk register`** section — one entry per `⚠️` row with a runnable **probe recipe** (mirror
   ADR 0024 §5): exact `omi` command on `:4002` + the observable that confirms/refutes + the fallback.
   Seed recipes:
   - *combo net-credit*: place a far-from-market credit spread, e.g.
     `omi option-combo --action sell --leg "SELL 1 AAPL <expiry> <lowStrike> C" --leg "BUY 1 AAPL <expiry> <highStrike> C" --qty 1 --limit -0.05` on `:4002` ⇒ `omi orders` shows the BAG accepted at the
     intended net credit ⇒ `omi cancel <id>`. Fallback if rejected/mispriced: record the sign convention
     Tiger actually requires.

5. **Deferred (D2)**: do NOT run the probes — they need a live US-session gateway. The doc ships with the
