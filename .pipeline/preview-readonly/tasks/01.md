# Card 01 — preview-readonly (`--preview` must never transmit)

Make `tests/order_preview_command.rs` GREEN. Read `PRD.md` + `arch.md` §Component changes + `CONTEXT.md`
+ ADR 0027 FIRST. **R1 REFUTED: Tiger TRANSMITS whatIf orders — the preview path must call NO
`place_order`.** This REPLACES the order-preview whatIf mechanism.

## Scope (impl-paths ONLY — do NOT touch `tests/`)

- **`src/ib/trade.rs`**:
  - **`shape_preview` — NEW signature (the old `(&Contract,&Order,&OrderState)` is gone):**
    `pub fn shape_preview(contract: serde_json::Value, order: &Order, multiplier: f64, notional_ccy: &str) -> Value`.
    Pure FROZEN seam. Envelope (exact keys, sorted): `action` (`{:?}` of `order.action`), `contract`
    (the passed-in JSON verbatim), `note` ("margin/commission unavailable on this gateway (whatIf
    transmits)"), `notional` = `order.limit_price.map(|l| order.total_quantity * l.abs() * multiplier)`
    (⇒ JSON `null` for MKT/None), `notional_currency` (= `notional_ccy`), `order` {type, qty, limit},
    `preview` (true), `transmits` (**false**). NO `what_if`/`margin`/`commission`/`status`.
  - **REMOVE `preview_with_client`** (the `place_order`+`what_if` path). Replace with a read-only fn,
    e.g. `preview_stk_option(&Client, ctx, &Contract, &Order) -> Result<Value, AppError>`:
    `client.contract_details(&contract)` (READ; mirror `src/ib/contract.rs`) → build the `contract`
    JSON {symbol, conid, sec_type, exchange, currency, long_name; option adds expiry/strike/right from
    the built Contract} → `shape_preview(contract_json, order, multiplier, currency)`. NO `place_order`.
    Multiplier: STK 1.0; option 100.0.
  - **Read-shaped gate — move the preview branch BEFORE `require_live_write_gate`.** In `place_core`
    (:471): `if cfg.preview { let client = connect(cfg)?; return preview_stk_option(&client, ctx, contract, order); }`
    THEN `require_live_write_gate(cfg)?` + connect + resolve_account + `place_with_client` for the real
    path. In `option_combo` (:820) and `option_close` (:1072): their conids are ALREADY resolved before
    the branch — the preview arm shapes from the resolved legs/contract (build the `contract` JSON with
    the resolved conids; combo multiplier 100 on `|net_limit|`, close multiplier 100) and must NOT hit
    `require_live_write_gate`. Keep the real arms unchanged (they still gate).
  - The `--preview` flag + `Config.preview` already exist (order-preview) — no cli/config change needed.
- **`src/ib/mod.rs`**: the `pub use trade::{… shape_preview …}` re-export is by NAME — unchanged unless
  you rename; adjust only if needed.
